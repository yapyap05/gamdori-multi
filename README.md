# 🎮 Arcade Gamdori 🎮
- 레트로 아케이드 감성의 게임 사이트 이고, 슬롯 머신의 느낌을 살렸습니다.

# 📌 프로젝트 개요
### 프로젝트명: Arcade Games
### 장르: 아케이드 / 캐주얼
### 플랫폼: Web (Desktop 기준, 모바일 부분 대응)
### 컨셉: 레트로 아케이드 머신 + 네온 UI + CRT 효과

# 🧱 전체 구조
```
/ (root)
 ├─ index.html            # 메인 허브 (게임 선택)
 ├─ xxx-game.html         # 제공 게임 화면
 ├─ style.css             # 공통 스타일 (네온, 글래스, CRT)
 ├─ xxx-game.js           # 제공 게임 로직
 └─ assets/
     ├─ 적용전/
     └─ 적용전/
```
# 🏠 index.html 분석 요약
- index.html은 아케이드 게임의 메인 허브 역할을 합니다.

## 주요 특징
- Bootstrap 기반 레이아웃
- Carousel을 활용한 게임 선택 UI
- 각 게임 카드마다 개별 배경 이미지 적용
- 네온 타이틀, 글래스 카드 스타일 사용

## 역할
- 사용자가 여러 미니 게임 중 하나를 선택
- 각 게임은 독립적인 HTML 페이지로 분리
- UI 스타일과 분위기를 통일하는 기준점

# 🌐 언어 시스템
- html, css, js

# 🎨 UI / UX 설계
- 좌 / 우 분할 구조
- 왼쪽 영역: 게임 플레이
- Stage / Time 표시
- 슬롯
- 입력창
- 오른쪽 영역: 정보 및 로그
- HUD (정답 / 오답 / 타임아웃 / 시도)
- 단계 규칙 도움말
- 문제 & 결과 로그 테이블

이 세 가지를 최우선 가치로 설계되었습니다.
